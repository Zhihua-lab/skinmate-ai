# 肤记 · SkinMate AI 护肤方案 App 前端

「肤记」是一个移动端 AI 护肤计划助手的前端 Demo：用户粘贴一条抖音视频链接，即可生成专属护肤方案，并通过 AI 对话不断优化、每日打卡记录皮肤变化、查看护肤排行榜。项目带有可爱的护肤小助手吉祥物形象，整体还原移动端 App 质感。

## 功能页面

- **首页 / 方案生成**：粘贴抖音视频链接生成护肤方案，提供热门案例入口
- **方案详情**：左右滑动查看「洁面 → 补水 → 精华 → 保湿」四步方案，含产品、价格、成分、用法，可收藏并查看总价预估
- **AI 修改方案**：对话式调整方案，支持快捷标签（减少精华 / 增加保湿 / 更换洁面），实时生成更新后的方案
- **每日打卡**：上传皮肤状态照片、日历打卡、连续天数统计与每日提醒
- **打卡记录详情**：查看当日照片、备注与 AI 建议
- **排行榜**：总榜 / 同方案榜 / 好友榜，展示坚持天数与我的排名
- **我的**：个人信息与方案、收藏、打卡记录、肤质档案、设置入口

## 运行方式

```bash
npm install
npm run dev
```

浏览器打开 Vite 输出的本地地址即可（支持局域网访问，可用手机扫码预览移动端效果）。

构建生产版本：

```bash
npm run build
npm run preview
```

## 技术栈

- React 18
- Vite
- lucide-react 图标
- 原生 CSS

## 说明

当前版本使用 mock 数据实现交互，不依赖后端。页面重点还原移动端 App 质感、卡片式布局、紫色 / 绿色 / 暖色主色体系，以及「方案生成 → 方案详情 → AI 修改 → 每日打卡 → 打卡记录 → 排行榜」的完整前端流程。
