# 肤记 · SkinMate AI 护肤方案 App 前端

> 让视频里的护肤经验，从"看过"变成"用上"。

「肤记」把抖音上优质的护肤内容，转化为一套**可执行、可复用、可坚持**的个人护肤能力。用户只需粘贴一条抖音视频链接，AI 就会把视频里零散的护肤建议，重构成一份结构清晰的专属方案——并支持对话调整、每日打卡执行、长期记录皮肤变化。

## 我们解决的问题

抖音上有大量优质的护肤视频，但内容看完往往就忘了：步骤记不住、产品搭配不清楚、更难长期执行。优质内容停留在"被看到"，却很难真正转化为对个体有用的能力。

「肤记」的思路是 **从"使用"出发**——让 AI 接住视频里的经验，把它变成一个轻量、可调用、可坚持的护肤工具：

- **重构**：把一条视频里分散的护肤建议，结构化为「洁面 → 补水 → 精华 → 保湿」的清晰步骤
- **可执行**：每一步都给出产品、价格、成分、具体用法与视频时间轴引用，方案落到可操作的细节
- **可调整**：通过 AI 对话按个人需求改写方案（如减少精华、增加保湿、控制预算）
- **可坚持**：每日打卡、皮肤状态记录与排行榜激励，让"用上"持续发生

## 功能页面

- **首页 / 方案生成**：粘贴抖音视频链接生成护肤方案，提供热门案例入口
- **方案详情**：左右滑动查看「洁面 → 补水 → 精华 → 保湿」四步方案，展示产品、价格、成分、用法与默认展开的视频时间轴；步骤内可单独更换产品，并支持展开总价明细
- **方案全局操作**：底部提供「保存方案 / AI 调整方案 / 开始打卡」三个清晰入口；分享卡片收纳在更多菜单与保存成功弹窗中
- **AI 调整方案**：对话式调整预算、产品、步骤与肤质适配，支持快捷标签（减少精华 / 增加保湿 / 更换洁面），实时生成更新后的方案
- **每日打卡**：上传皮肤状态照片、日历打卡、连续天数统计与每日提醒
- **打卡记录详情**：查看当日照片、备注与 AI 建议
- **排行榜**：总榜 / 同方案榜 / 好友榜，展示坚持天数与我的排名
- **我的**：个人信息与方案、收藏、打卡记录、肤质档案、设置入口

## 体验流程

```
粘贴抖音视频链接 → AI 重构护肤方案 → 查看方案 → 保存 / AI 调整 / 开始打卡 → 记录变化 → 排行榜激励
```

## 运行方式

```bash
npm install
npm run dev
```

浏览器打开 Vite 输出的本地地址即可（支持局域网访问，可用手机扫码预览移动端效果）。

构建生产版本：

```bash
npm run build
npm run preview
```

当前版本使用 mock 数据实现交互，聚焦于交互体验与产品价值的完整呈现。页面延续粉色主视觉、圆角卡片与轻量阴影，强调真实移动端 App 的信息密度和操作层级，并打通「方案生成 → 方案详情 → AI 调整 → 保存方案 / 开始打卡 → 打卡记录 → 排行榜」的完整前端流程。
