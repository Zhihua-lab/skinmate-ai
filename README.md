# AI 护肤方案 App 前端

这是根据 6 张设计图还原的移动端 React 前端 Demo，包含完整页面流程：

- 首页 / 方案生成页
- 方案详情页
- AI 对话与修改方案页
- 打卡页面
- 打卡记录详情页
- 排行榜页面
- 我的页面占位

## 运行方式

```bash
npm install
npm run dev
```

浏览器打开 Vite 输出的本地地址即可。

## 技术栈

- React
- Vite
- lucide-react 图标
- CSS 原生样式

## 说明

当前版本使用 mock 数据实现交互，不依赖后端。页面重点还原移动端 App 质感、卡片式布局、紫色/绿色主色体系，以及“方案生成 → 方案详情 → AI 修改 → 打卡记录 → 排行榜”的完整前端流程。
